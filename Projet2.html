Projet : Site web pour l'Association Nou Lé Là

J'ai conçu un site web pour l'Association Nou Lé Là, qui réunit la communauté réunionnaise à Béziers autour de la culture créole. Le site a été développé avec HTML et CSS en utilisant Visual Studio Code. Mon objectif était de créer une plateforme conviviale pour les membres, avec une navigation fluide et un design dynamique. Les fonctionnalités incluent : Une bannière dynamique pour capter l'attention dès l'arrivée sur la page Un carrousel d'images pour mettre en avant les événements de l'association Des liens externes pour rediriger les visiteurs vers des partenaires et des ressources Des boutons interactifs pour un accès facile aux différentes sections Une galerie photos pour partager les moments forts de l'association Le site est entièrement responsive, offrant une expérience optimale sur mobile et desktop.
